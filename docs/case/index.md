# Case / Cas

This section groups the case studies from your PDF into cleaner practice pages.

## Included cases

1. Importation d’électricité des États-Unis vers le Québec
2. Enquête sur un accident du travail dans une usine
3. Déclassement des installations nucléaires
4. Prévision et exploitation d’un nouveau forage pétrolier
5. Faisabilité d’une passerelle piétonne et cycliste

## How to use these pages

- Read the **Contexte**
- Identify the **enjeux / key issues**
- Draft a response with:
  - introduction
  - 2–3 main points
  - recommendations
  - concise conclusion

## Suggested response structure

### Version courte

> Le projet vise à… Cependant, il présente plusieurs défis. Tout d’abord, … Ensuite, … Enfin, … Pour conclure, il est recommandé de…

### Version plus formelle

> Dans le cadre de ce projet, plusieurs enjeux techniques, environnementaux et organisationnels doivent être pris en compte. D’une part, … D’autre part, … Par conséquent, il est recommandé de mettre en place…

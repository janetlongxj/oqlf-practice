# Writing Frameworks / Cadres d’écrit

## 1) General response structure

### Version simple

> Le projet vise à… Cependant, il présente plusieurs défis. Tout d’abord, … Ensuite, … Enfin, … Pour conclure, il est recommandé de…

### Version plus soutenue

> Dans le cadre de ce projet, plusieurs enjeux techniques, environnementaux et organisationnels doivent être pris en compte. D’une part, … D’autre part, … Par conséquent, il est essentiel de mettre en place…

## 2) Problem–solution pattern

> Le principal problème concerne…  
> Pour y répondre, il convient de…  
> Cette mesure permettrait de…

## 3) Stakeholder-focused pattern

> Plusieurs parties prenantes sont concernées par ce dossier, notamment…  
> Il est donc important d’assurer une communication transparente afin de…

## 4) Short conclusion formulas

- En conclusion, une planification rigoureuse est essentielle.
- Pour conclure, ces mesures permettront de réduire les risques.
- En somme, le projet peut être réalisé de façon sécuritaire et efficace.
- Ainsi, une approche intégrée favorisera la réussite du projet.

## 5) Useful correction notes

Prefer:
- **il faut / il convient de / il est recommandé de**
- **présente plusieurs défis**
- **mettre en place**
- **assurer la conformité**
- **réduire les risques**

Avoid overly literal patterns when possible:
- *la manque de budget* → **le manque de budget**
- *la risque* → **le risque**
- *transformer à* → **transformer en**
- *surveilleraient régulièrement* → **surveilleraient régulièrement le site** / **assureraient un suivi régulier**
